Firefox Marketplace Frontend Documentation
==========================================

In-depth developer documentation for `Firefox Marketplace`_. frontend projects.

.. _`Firefox Marketplace`: https://marketplace.firefox.com/

Marketplace frontend projects all share a common framework. Current projects
include:

* `Marketplace Frontend <https://github.com/mozilla/fireplace>`_
* `Marketplace Template <https://github.com/mozilla/marketplace-template>`_
* `Marketplace Curation Tools <https://github.com/mozilla/transonic>`_
* `Marketplace Communication Dashboard <https://github.com/mozilla/commbadge>`_
* `Marketplace Operator Dashboard <https://github.com/mozilla/marketplace-operator-dashboard>`_
* `Marketplace Statistics <https://github.com/mozilla/marketplace-stats>`_

Pieces that make up the framework include:

* `Marketplace Core Modules <https://github.com/mozilla/marketplace-core-modules>`_
* `Marketplace Gulp <https://github.com/mozilla/marketplace-gulp>`_
* `Marketplace Node Modules <https://github.com/mozilla/commonplace>`_
